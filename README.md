## Installation

```
yarn 
```
or
```
npm i
```

## Running tests

```
yarn test
```
or
```
npm test
```

## Updating snapshots
```
yarn test:update
```
or
```
npm run test:update
```

**Note** requires node >= 6.9x & npm >= 3
